# Data Professional Survey Dashboard — Power BI

## Overview
An interactive Power BI dashboard analyzing survey responses from 600+ data professionals worldwide. The dashboard explores salary trends, job satisfaction, favorite tools and programming languages, and how difficult people found breaking into the data field.

## Objective
To practice the full Power BI workflow — from importing and cleaning raw survey data to building an interactive, decision-ready dashboard — using real survey data instead of a toy dataset.

## Tools Used
- **Power BI Desktop** — data modeling, DAX measures, visualization
- **Power Query** — data cleaning and transformation

## What the Dashboard Shows
- Breakdown of survey takers by **country**
- **Average salary** by job title (Data Scientist, Data Analyst, Data Engineer, etc.)
- **Job satisfaction** and **happiness scores** using gauge visuals
- Favorite **programming languages** used by data professionals
- Salary comparison **by gender**

## Key Insights
- Salary varies significantly by job title, with Data Engineers and Data Scientists trending higher than Data Analysts
- Job satisfaction scores cluster in the moderate range, suggesting room for improvement industry-wide
- Python and SQL emerged as the most commonly used tools among respondents
